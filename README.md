# 🛡️ HackTech Network Architect PRO
**Infraestructura de Red y Seguridad lógica para especialistas de élite.**

Esta aplicación es la herramienta oficial de **Hacktechnology** para el diseño, despliegue y documentación de redes segmentadas (VLANs). Optimizada para flujos de trabajo con **Termux** y dispositivos móviles.

---

## 🚀 Funcionalidades Principales
- **Generador de Scripts CLI:** Configuración automática para MikroTik (RouterOS) y Switch Cisco/TP-Link.
- **Calculadora de Subredes:** Validación instantánea de Gateways e IPs de Host.
- **Seguridad Administrativa:** Generador de contraseñas de alta entropía.
- **Reporte Técnico Digital:** Generación de PDF profesional con **Código QR** integrado para el cliente.

---

## 📊 Estándar de Configuración (VLAN Maestro)
Para garantizar la uniformidad en todas las instalaciones de **HackTech**, se debe respetar el siguiente esquema:

| ID VLAN | Segmento IP | Propósito |
| :--- | :--- | :--- |
| **10** | `192.168.10.x` | Gestión (Switches/Antenas) |
| **20** | `192.168.20.x` | CCTV (Cámaras/NVR) |
| **30** | `192.168.30.x` | Datos (Oficina/Wi-Fi) |

---

## 🛠️ Guía de Uso para Técnicos
1. **Configuración:** Ingresa los parámetros de red solicitados en la App.
2. **Despliegue:** - Haz clic en **Copiar Script**.
   - Abre **Termux** y accede al equipo vía SSH o consola.
   - Pega el código generado.
3. **Documentación:**
   - Presiona **Generar Reporte PDF**.
   - Guarda el archivo y envíalo al grupo de soporte de la empresa.
   - Si es posible, pega el código QR en el rack del cliente.
